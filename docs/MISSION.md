# Mission

Build a **Loyalty Management Platform** — a SaaS serverless application on AWS that lets businesses run loyalty programs (points, tiers, rewards, campaigns). Deploy with AWS CDK. Support multi-tenant usage, APIs, and a web dashboard.

Progress is measured by:

- Passing tests (unit and integration)
- Merged feature branches (via git-flow: `feature/*` → `develop`)
- Updated documentation (PRD, ARCHITECTURE, PROGRESS)

All agents and contributors should align to [docs/PRD.md](PRD.md) and [docs/ARCHITECTURE.md](ARCHITECTURE.md).
